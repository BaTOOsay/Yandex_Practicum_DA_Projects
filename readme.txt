# Проекты курса "Аналитик данных" от Яндекс Практикума

Успешно прошёл курс по анализу данных от Яндекс Практикума, в ходе которого создавал проекты, направленные на проработку навыков, необходимых для погружения в профессию. Описания каждого проекта отражено в самих проектах, ознакомление с проектами по SQL - по запросу (чтобы не выкладывать решения).
